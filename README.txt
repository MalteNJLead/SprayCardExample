Spray Card Example Project.

About the project: 
A paint manufacturer needs to test paints regularly. In this case the testing procedure is conducted like 

The robot is equipped with a spraypainter and required to paint lines on a card(wall) with a certain amount of overlap. 
A key feature is that the movement starts before the spraygun fires, which means the movement is unblocked. 
Makes use of the commonLib Library.
